# Mensagem para enviar ao cliente

Segue o link para teste do sistema de Proposta de Locacao Residencial:

```text
https://pontualpropostasdelocacao.streamlit.app
```

Como testar:

1. Abra o link.
2. Escolha o tipo de locacao: Residencial, Comercial ou Temporada.
3. Preencha os dados do imovel e do proponente.
4. Se necessario, escolha `Com fiador` e preencha os dados adicionais.
5. Anexe os documentos solicitados.
6. Informe o canal de retorno: WhatsApp ou e-mail.
7. Marque a declaracao final.
8. Clique em `Gerar proposta em PDF`.
9. Baixe o PDF ou o ZIP completo com os documentos.

Esta e uma versao de demonstracao para validar o fluxo. A versao final pode incluir painel administrativo, historico de propostas, login, envio por e-mail e armazenamento seguro dos documentos.
